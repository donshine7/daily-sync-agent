# daily-sync-agent
이메일, SNS, 음성 파일 등 파편화된 비정형 데이터를 분석하여 스마트하게 일정을 자동화하는 멀티모달 AI 에이전트

🚀 Overview
daily-sync-agent는 매일 여러 채널을 통해 쏟아지는 방대한 소통 기록을 하나의 정형화된 캘린더 이벤트로 동기화(Sync)하는 지능형 백그라운드 에이전트입니다.

텍스트 기반의 이메일과 SNS 메시지는 물론, 음성 파일까지 입력값으로 받아들여 맥락을 이해합니다. 수집된 다채널 데이터를 기반으로 사용자의 개입 없이 빠르고 정확하게 실행 가능한 일정(Actionable Schedule)을 생성하여 생산성을 극대화합니다.

✨ Key Features
다중 매체 수집 및 전처리 (Multimodal Ingestion)

이메일(IMAP/API) 및 SNS 메신저 텍스트 파싱.

오프라인 및 온디바이스 환경에서도 지연 시간(Latency)을 최소화하여 빠르고 가볍게 동작할 수 있도록 최적화된 오디오 STT(Speech-to-Text) 파이프라인.

지능형 맥락 분석 (Intelligent Extraction)

LLM을 활용하여 복잡하고 비정형화된 대화 속에서 날짜, 시간, 장소, 참석자, 핵심 안건 등 캘린더 생성에 필요한 필수 메타데이터를 정확하게 추출.

자동화된 캘린더 동기화 (Automated Calendar Sync)

추출된 데이터를 표준 캘린더 포맷으로 변환하고 Google Calendar 등 외부 API와 연동하여 사용자의 일정표에 즉각적으로 반영.

🛠️ Typical Use Cases
이동 중 남긴 음성 메모나 통화 녹음을 바탕으로 자동으로 회의 일정 잡기

받은 편지함에 섞여 있는 세미나, 항공권, 미팅 안내 메일을 캘린더에 자동 등록

SNS 메신저로 주고받은 모임 약속을 감지하여 일정 초안 생성
